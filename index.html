About

ターゲットは社会保険労務士・行政書士・税理士・介護事業所・建設業者など、 書類業務が多く・IT化が遅れている日本の中小企業・士業事務所。

技術スタック: Python / Streamlit / Supabase / Stripe / Google Gemini API / GitHub Actions
